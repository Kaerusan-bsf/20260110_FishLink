# FishLink MVP Spec (Minimal)

## Screens (5)
0. Farmer Listing
1. Today’s Farms
2. Farm Detail
3. Farmer Accept / Reject / Ready
4. Request Status

## Status Flow
Requested → Accepted → Preparing → Ready → Completed

## Do NOT implement (MVP)
- Chat
- GPS realtime tracking
- Meeting point
- Photos in list
- Minute-level time selection
- Price negotiation
